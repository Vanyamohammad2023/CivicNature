# 🌿 CivicNature — Report. Track. Improve.
 
> **A civic reporting platform empowering citizens of Sulaymaniyah to document urban issues and nature sightings — transparently, publicly, and for real change.**
 
🔗 **Live Demo:** [extraordinary-bublanina-36b200.netlify.app](https://extraordinary-bublanina-36b200.netlify.app/)
 
---
 
## 📖 About
 
CivicNature is a web application that bridges the gap between citizens and city management. Residents of Sulaymaniyah can report broken infrastructure, environmental damage, and rare wildlife sightings — all on one public platform. Every report is tracked from submission to resolution, giving the community full transparency and a real voice.
 
Built in alignment with the **United Nations SDG 11 — Sustainable Cities & Communities**.
 
---
 
## ✨ Features
 
| Feature | Description |
|---|---|
| 🏙️ **Urban Issue Reporting** | Report potholes, broken lights, illegal dumping, water failures, and more |
| 🌿 **Nature Sighting Reporting** | Document rare species, wildlife, and protected plants with a dedicated mode |
| 📸 **Photo Uploads** | Attach photos from camera or gallery for undeniable visual proof |
| 📊 **Public Dashboard** | Filter, search, and sort all reports by category, date, votes, or severity |
| 🗺️ **Interactive Map** | View all reports pinned on a Sulaymaniyah city map with urban/nature colour coding |
| 👍 **Community Upvoting** | Neighbours vote on what matters most — urgency rises from the community |
| 🦋 **Species Guide** | Info on 6 protected species found around Sulaymaniyah (status, habitat, scientific names) |
| 🔄 **Status Tracking** | Every report shows Pending → In Progress → Resolved, updated in real time |

## 🌍 Context
 
Sulaymaniyah, Kurdistan Region of Iraq, faces ongoing urban challenges — potholes, lighting failures, drainage issues, illegal dumping — alongside environmental threats to rare highland species like the Wild Tulip and Eastern Imperial Eagle.
 
CivicNature was built to give residents a structured, transparent channel to document these problems and hold authorities accountable — no account needed, no barriers.
